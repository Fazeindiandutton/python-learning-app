Variables & Data Types

In Python, variables are created when you assign a value to them. There's no need to declare a variable type explicitly.

Creating Variables

Example 
name = "Alice"                    # String
age = 25                          # Integer
height = 5.8                       # Float
has_dog = True                     # Boolean

print(f"{name} is {age} years old")
print(f"Height: {height} feet")

Data Types

  • int - Integers (25, -3, 0)
  • float - Floating point numbers (3.14, -2.5)
  • str - Strings ("hello", 'world')
  • bool - Booleans (True, False)
  • list - Lists [1, 2, 3]
  • dict - Dictionaries {"key": "value"}
  • tuple - Tuples (1, 2, 3)
  • set - Sets {1, 2, 3}

Control Flow

Control flow statements let you control the order in which your code executes.

Conditional Statements

Example 
age = 18

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# Multiple conditions
count = 5

if count > 0:
    print("Positive")
elif count < 0:
    print("Negative")
else:
    print("Zero")

Loops

While Loop 
count = 5
while count > 0:
    print(f"Count: {count}")
    count -= 1
For Loop 
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Range loop
for i in range(5):
    print(f"Number: {i}")

Functions

Functions are reusable blocks of code that perform specific tasks.

Defining Functions

Example 
def greet_user(name, greeting="Hello"):
    return f"{greeting}, {name}!"

# Function call
message = greet_user("Alice")
message2 = greet_user("Bob", greeting="Hi")
print(message)
print(message2)

# Function with return value

def calculate_rectangle_area(width, height):
    area = width * height
    print(f"Area: {area}")
    return area

square_area = calculate_rectangle_area(5, 5)
rectangle_area = calculate_rectangle_area(4, 6)

Docstrings

Example 
def add_numbers(a, b):
    """
    Add two numbers and return their sum.
    
    Parameters:
    a (int/float): First number
    b (int/float): Second number
    
    Returns:
    int/float: The sum of a and b
    """
    return a + b

Objects & Classes

Python uses object-oriented programming (OOP) to organize code into classes and objects.

Creating Classes

Example 
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, my name is {self.name}"
    
    def birthday(self):
        self.age += 1
        return f"Happy birthday! New age: {self.age}"

# Creating objects
person1 = Person("Alice", 25)
person2 = Person("Bob", 30)

print(person1.greet())
print(person2.greet())
print(person1.birthday())
print(person2.greet())

Built-in Methods

String Methods
